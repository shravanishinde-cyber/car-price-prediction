## car-price-prediction
Machine Learning project to predict car prices using Python.
## Overview
This project predicts the price of used Audi cars using Machine Learning.
## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook
## Dataset
Audi Car Dataset
## Data Preprocessing
- Checked missing values
- Removed duplicate records
- Performed data cleaning
- Encoded categorical variables
## Machine Learning
- Trained regression models
- Evaluated performance using R² Score and MAE

- Car Price Prediction Project.ipynb
